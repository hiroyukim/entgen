# [WIP] entgen

Generate Ent schemas

## Usage

```
$ ./entgen -dirver postgres -dsn "postgres://..."
```

## Supported Drivers

|Name    |Driver    |
|--------|----------|
|postgres|PostgreSQL|
|sqlite3 |SQLite3   |
|mysql   |MySQL     |

## Installation

```
go install github.com/mattn/entgen@latest
```
